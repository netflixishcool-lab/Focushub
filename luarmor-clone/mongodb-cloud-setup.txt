MONGODB ATLAS CLOUD SETUP (100% KOSTENLOS!)
=============================================

Schritt 1: Konto erstellen
--------------------------
1. Gehe zu: https://www.mongodb.com/cloud/atlas
2. Klick "Try Free"
3. Registriere dich (Email + Passwort)
4. Verifiziere deine Email

Schritt 2: Cluster erstellen
---------------------------
1. Nach Login: "Create Deployment"
2. Waehle "M0" (Kostenlos!) ✓
3. Region: z.B. Frankfurt (Europe)
4. Klick "Create Cluster"
5. Warte 3-5 Minuten (bekommt gruenen Haken)

Schritt 3: Connection String
----------------------------
1. Klick "Connect" Button
2. Waehle "Connect your application"
3. Waehle "Node.js" und neueste Version
4. Kopiere den Connection String:

   mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/focushub-admin

5. Ersetze:
   - username = Dein Benutzername
   - password = Dein Passwort
   - cluster0.xxxxx = Dein Cluster Name

Schritt 4: In .env eintragen
-----------------------------
Oeffne .env und aendere MONGODB_URI:

MONGODB_URI=mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/focushub-admin

FERTIG! Deine Cloud-DB laeuft! Cloud-Icon

FEHLERBEHEBUNG:
===============

Problem: "MongoDB connection failed"
Loesung:
  1. Gehe zu MongoDB Atlas
  2. Kopiere Connection String neu
  3. Ersetze username/password korrekt
  4. Stelle sicher dass Cluster laeuft (gruen)

Problem: "IP not whitelisted"
Loesung:
  1. Gehe zu MongoDB Atlas
  2. Gehe zu "Security" -> "Network Access"
  3. Addiere IP 0.0.0.0/0 (alle IPs erlauben)

Problem: "Authentication failed"
Loesung:
  - Ueberprüfe username und password
  - Stelle sicher dass kein @ Symbol im Password
  - Wenn @ im Password: URL-encode (z.B. %40)


SICHERHEIT:
===========

Wichtig fuer Production:
- Verwende STARKE Passwoerter
- Nutze IP Whitelist (nicht 0.0.0.0/0)
- Aktiviere 2FA auf MongoDB Atlas
- Regelmaeßig Backups machen
- Nie Connection String in Code pushen!
